# Privacy Policy

Last updated: April 19, 2026

## Overview

Personal Notes for LinkedIn lets you save private notes on LinkedIn profiles inside your own browser.

This extension does not operate any backend service and does not send your notes to the developer or any third-party server.

## What Data Is Stored

The extension stores the following data in your browser:

- Notes you write for LinkedIn profiles
- Profile identifiers used to associate a note with a profile, such as username or member ID
- Profile display name when available
- Note metadata such as creation and last-updated timestamps
- Optional local file handle metadata if you enable file sync in supported browsers

## How Data Is Used

Your data is used only to provide the extension’s features, including:

- Showing your saved notes on LinkedIn profile pages
- Displaying notes in the notes viewer
- Exporting and importing notes
- Writing notes to a local sync file when you enable that feature

## Where Data Is Stored

Notes are stored using your browser’s extension storage APIs.

If you are signed in to browser sync, your browser may sync this data across your own devices using your browser provider’s sync service. This extension does not control that sync service.

If you use the optional file sync feature, a plain-text JSON copy of your notes is written to the local file you choose.

## Data Sharing

This extension does not sell, rent, transmit, or share your notes with the developer or with third parties.

The extension does not include analytics, advertising, tracking pixels, or telemetry.

## Permissions

The extension requests the `storage` permission to save your notes and related metadata.

The extension runs on LinkedIn pages so it can display and edit notes on profile pages.

## Data Security

Notes are stored in plain text and are not encrypted by the extension.

Anyone with access to your browser profile, synced browser data, or exported/synced JSON files may be able to read your notes.

## Your Choices

You can control your data at any time by:

- Editing or deleting notes in the extension
- Exporting your notes for backup
- Removing exported or synced JSON files you created
- Uninstalling the extension
- Clearing extension storage through your browser
